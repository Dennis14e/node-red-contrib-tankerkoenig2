# node-red-contrib-tankerkoenig2

Node-RED nodes to communicate with the API of [Tankerkoenig.de](https://creativecommons.tankerkoenig.de/).


## Available nodes
![Nodes](./docs/nodes.png)

- Tankerkoenig Radius
- Tankerkoenig Prices
- Tankerkoenig Detail
- Tankerkoenig Complaint
